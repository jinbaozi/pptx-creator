# Editability Ladder

- Level 5: Fully native editable.
- Level 4: Mostly editable.
- Level 3: Semi-editable.
- Level 2: Low editability.
- Level 1: Raster fallback.

M1.1 target:

- Generated content: Level 4-5.
- HTML conversion manifests: Level 3-5.
- Image replica manifests: Level 2-4.

A full-slide background image with zero editable text must be reported as Level 1.
