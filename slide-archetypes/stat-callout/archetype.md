# stat-callout

This archetype defines a single hero metric plus supporting text (one big number per slide). It should be selected by `slide-design-specs.json` through the `layoutType` field when the slide role is "evidence" or "headline KPI".

Use this layout when a single number is the entire point of the slide. Multi-metric rows belong in the numbers-row content pattern (which composes stat-callout horizontally).
