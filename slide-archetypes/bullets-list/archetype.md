# bullets-list

This archetype defines a single-column bulleted list of points (4-7 items). It should be selected by `slide-design-specs.json` through the `layoutType` field when the slide role is "supporting points" or "key takeaways".

Use this layout when a single claim needs 4-7 supporting points that share the same weight. No icons are required; if used, every bullet must carry the same icon treatment.
