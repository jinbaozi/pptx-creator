# icon-grid

This archetype defines a grid of icon-plus-label tiles, typically arranged 3x2 or 2x4. It should be selected by `slide-design-specs.json` through the `layoutType` field when the slide role is "feature list" or "capability overview" with parallel items.

Use this layout when 6-8 items share equal weight and each fits a single icon-label pair. If items need longer descriptions, fall back to a bullets-list or two-column.
