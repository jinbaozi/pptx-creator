# Cover archetype — role-aware rules

- **Headline claim**: One clear claim in the title using `{typography.title}`; floor `fontSize >= 36pt`, `lineHeight >= 1.05`.
- **Sub-claim**: One supporting line of body copy using `{typography.subtitle}`; floor `fontSize >= 14pt`, `lineHeight >= 1.30`.
- **Hero element**: Exactly one hero element (logo, accent rule, or single accent shape) is permitted. No secondary hero competing for attention.
- **Whitespace**: Page margins `>= {spacing.lg}`; vertical breathing room between hero and headline `>= {spacing.xl}`.
- **Card spacing**: Cover has no content cards; if decorative cards are introduced, gap `>= {spacing.md}`.
- **Color discipline**: Use `{colors.primary}` plus at most one accent or one muted surface; total accents `<= 2`.
- **Font-family cap**: Use `font-family` from `{typography.title}` and `{typography.body}` only; `<= 2` distinct families across the cover.
- **Decorative density**: At most one rounded-token variance and no icon-circle triad; cover must stay visually quiet.
- **Word mark**: Company / product wordmark kept to a single line, `fontSize >= 11pt`, never full-slide rasterized.
- **Editable target**: Aim for Editability Ladder Level 5 (fully native); never drop below Level 4 (text + main shapes editable).
