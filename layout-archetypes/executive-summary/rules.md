# Executive-summary archetype — role-aware rules

- **Headline claim**: One executive statement using `{typography.title}`; floor `fontSize >= 26pt`, `lineHeight >= 1.10`.
- **Supporting paragraph**: Lead paragraph uses `{typography.subtitle}`; floor `fontSize >= 14pt`, `lineHeight >= 1.35`.
- **Bullet body**: Supporting bullets use `{typography.body}`; floor `fontSize >= 11pt`, `lineHeight >= 1.45`.
- **Bullet count**: At most `5` primary bullets; secondary bullets must be inlined or moved to a backup slide.
- **Card spacing**: Gap between bullet rows `>= {spacing.sm}`; gap between bullet groups (e.g., "so what") `>= {spacing.md}`.
- **Whitespace**: Page margins `>= {spacing.lg}`; headline to first paragraph `>= {spacing.md}`.
- **Color discipline**: Use `{colors.primary}` plus at most one accent (highlighted callout); total accents `<= 2`. No per-bullet color cycling.
- **Font-family cap**: Distinct `font-family` count `<= 2` (one for headline, one for body) across the summary.
- **Decorative density**: At most one rounded-token variance; no icon-circle triad — keep executives focused on text.
- **One page, one message**: If two ideas compete, split the slide. The archetype refuses dual-thesis pages.
- **Editable target**: Native text boxes only; aim for Editability Ladder Level 5.
